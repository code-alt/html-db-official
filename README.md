  # Docs 
  
  ---
  
  There are 6 functions in HTML-DB:
  
  all() - gets all the data
  
  createTable() - creates a table
  
  getTable() - gets a table
  
  hasTable() - if it has the table return true or false
  
  toJSON() - turns it into JSON text
